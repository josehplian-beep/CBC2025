# Chin Bethel Church Website

A modern church website built with React, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
npm install
npm run dev
```

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS
- Supabase (Backend)
- Vite (Build tool)
- shadcn-ui (UI Components)
